<h2><a href="https://www.geeksforgeeks.org/problems/minimize-the-sum-of-product1525/1?page=2&category=Arrays,Java&difficulty=Easy&status=unsolved&sortBy=submissions">Minimize the sum of product</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-family: arial, helvetica, sans-serif;"><span style="font-size: 18px;">Given two arrays <strong>a[]</strong> and <strong>b[]</strong> of size <strong>n</strong> containing positive integers. Rearrange the elements of both arrays so that the value of: </span></span><span style="font-family: arial, helvetica, sans-serif;"><span style="font-size: 18px;">a[0] * b[0] + a[1] * b[1] + ... + a[n-1] * b[n-1] </span></span><span style="font-family: arial, helvetica, sans-serif;"><span style="font-size: 18px;">becomes minimum. </span></span><span style="font-family: arial, helvetica, sans-serif;"><span style="font-size: 18px;">Each element of a[] and b[] must be used exactly once.</span></span></p>
<p><span style="font-family: arial,helvetica,sans-serif;"><span style="font-size: 18px;"><strong>Examples:</strong></span></span></p>
<pre><span style="font-family: arial, helvetica, sans-serif;"><span style="font-size: 18px;"><strong>Input</strong>: a[] = [3, 1, 1], b[] = [6, 5, 4]<strong>
Output: </strong>23<strong>
Explanation:</strong> After rearranging: a[] = [1, 1, 3] and b[] = [6, 5, 4]. Minimum sum = (1 * 6) + (1 * 5) + (3 * 4) = 6 + 5 + 12 = 23.</span></span></pre>
<pre><span style="font-family: arial,helvetica,sans-serif;"><span style="font-size: 18px;"><strong>Input: </strong>a[] = [6, 1, 9, 5, 4] , b[] = [3, 4, 8, 2, 4]
<strong>Output: </strong>80
<strong>Explanation: </strong></span></span><span style="font-family: arial, helvetica, sans-serif;"><span style="font-size: 18px;">After rearranging: a[] = [1, 4, 5, 6, 9] and b[] = [8, 4, 4, 3, 2]. Minimum sum = (1 * 8) + (4 * 4) + (5 * 4) + (6 * 3) + (9 * 2) = 8 + 16 + 20 + 18 + 18 = 80.</span></span></pre>
